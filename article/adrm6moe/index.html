Java8

2024-04-16

简介

Java8 为Java语言,编译器,类库,开发工具与JVM带来了大量新特性;

  • 速度更快
  • 代码更少(Lambda表达式)
  • Stream API
  • 便于并行
  • 最大化减少空指针异常; Optional类
  • Nashorn 引擎, 允许JVM上运行JS应用

并行流和串行流

并行流就是把一个内容分为多个数据块, 并用不同的线程分别处理每个数据块的流, 可以很大程度上提高程序的执行效率

串行流 就是一个线程上,从上到下运行程序

Lambda 表达式

是一个匿名函数, 可以把Lambda表达式理解为一段可以传递的代码(将代码像数据一样进行传递); 使用Lambda表达式可以写出更简洁更灵活的代码.

本质: 就是接口的实例

举例

由匿名内部类转换到Lambda表达式;

实例1
@Test
public void test1() {
    Runnable r1 = new Runnable() {
        @Override
        public void run() {
            System.out.println("我爱北京天安门");
        }
    };
    r1.run();
    System.out.println("***********Lambda表达式*************");
    Runnable r2 = () -> System.out.println("我爱北京故宫");
    r2.run();
}
实例2
@Test
public void test2() {
    Comparator<Integer> com1 = new Comparator<Integer>() {
        @Override
        public int compare(Integer o1, Integer o2) {
            return Integer.compare(o1, o2);
        }
    };
    int compare1 = com1.compare(12, 21);
    System.out.println(compare1);
    System.out.println("***********Lambda表达式*************");
    Comparator<Integer> com2 = (o1, o2) -> Integer.compare(o1, o2);
    int compare2 = com2.compare(32, 21);
    System.out.println(compare2);
    // 方法引用的写法
    Comparator<Integer> com3 = Integer::compare;
    int compare3 = com3.compare(21, 21);
    System.out.println(compare3);
}

基本使用

符号含义

  • ->: Lambda操作符 或 箭头操作符
  • 左边: Lambda形参列表 (即接口中的抽象方法的形参列表)
  • 右边: Lambda体 (即重写的抽象方法的方法体)

使用(六种情况)

方式1

格式一：无参，无返回值

@Test
public void test1() {
    Runnable r1 = new Runnable() {
        @Override
        public void run() {
            System.out.println("我爱北京天安门");
        }
    };
    r1.run();
    System.out.println("**************Lambda表达式*************");
    Runnable r2 = () -> System.out.println("我爱北京故宫");
    r2.run();
}
方式2

格式二：Lambda 需要一个参数，但是没有返回值

@Test
public void test3() {
    Consumer<String> con = new Consumer<String>() {
        @Override
        public void accept(String s) {
            System.out.println(s);
        }
    };
    con.accept("Lambda表达式的本质是什么呢?");
    System.out.println("**************Lambda表达式*************");
    Consumer<String> con1 = (s) -> System.out.println(s);
    con1.accept("Lambda表达式的本质就是: 作为接口的实例");
}
